# 物体モデル設計

## 概要

シミュレーションで使用する物体の接触モデル（弾塑性遷移、接触切替え）と5種の物体パラメータの決め方を定める。

## 設計判断

### 判断1: 弾塑性遷移モデル — 指数減衰

**問題**: 物体剛性 K_obj(θ) の非線形性をどうモデル化するか

**選択肢**:
1. 指数減衰: K_obj(θ) = K_0 × exp(-β(θ - θ_yield))  (θ ≥ θ_yield)
2. 区分線形: K_obj(θ) = K_0 × (1 - α(θ - θ_yield))  (θ ≥ θ_yield)
3. 実測データをテーブル参照

**決定**: 選択肢1 — 指数減衰

**理由**:
- 滑らかで微分可能。RFOB（微分ベース推定器）が捕捉しやすい
- research_roadmap（Step 2-3）で提案されたモデルと一致
- 材料力学的にも降伏後の剛性低下は指数的減衰に近い実験報告がある
- パラメータが2つ（K_0, β）だけで直感的に調整しやすい

**トレードオフ**:
- **利点**: 滑らか、パラメータが少ない、微分可能
- **欠点**: 実物体の正確な再現ではない（だがシミュレーションの目的は理論検証）

**数式**:
```
         ┌ K_0                              (θ_rel < θ_yield)
K_obj = ─┤
         └ K_0 × exp(-β × (θ_rel - θ_yield))  (θ_rel ≥ θ_yield)

τ_ext = K_obj(θ_rel) × θ_rel
```

- K_0 [Nm/rad]: 弾性域の剛性（一定）
- θ_yield [rad]: 降伏点（弾性→塑性の遷移開始角度）
- β [1/rad]: 減衰率（大きいほど急激に軟化）

### 判断2: 接触モデル — 角度閾値で切替え

**問題**: 自由空間（物体に触れていない）と接触状態の切替えをどうモデル化するか

**選択肢**:
1. 角度閾値で切替え: θ < θ_contact なら τ_ext=0、θ ≥ θ_contact なら τ_ext=K_obj×(θ-θ_contact)
2. 漸近的接触: τ_ext = K_obj × max(0, θ - θ_contact)（数学的に同一）

**決定**: 選択肢1 — 角度閾値で切替え（実装上は選択肢2と等価）

**理由**:
- 単純明快。θ_contact はconfig で物体ごとに指定
- 実機でも接触は角度（位置）で概ね決まる
- max(0, ...) で実装すれば分岐なしで滑らか

**実装**:
```cpp
double theta_rel = theta - theta_contact;
if (theta_rel < 0) theta_rel = 0;  // 非接触
double tau_ext = K_obj(theta_rel) * theta_rel;
```

**注意**: シミュレーション中の `x₁ = θ_rel` は接触後の相対角度。接触前は x₁ = 0 として扱う。

### 判断3: 物体パラメータ — 概算値で開始、後に実測値に置換

**問題**: 5種の物体（空アルミ缶、PETボトル水入り、PETボトル空、紙コップ空、紙コップ水入り）のパラメータをどう決めるか

**選択肢**:
1. 文献値・常識的な値で概算し、後で実測値に置換
2. 先に実物体を測定してからシミュレーション
3. 任意のテスト値（現実との対応は気にしない）

**決定**: 選択肢1 — 概算値で開始

**理由**:
- シミュレーション（Step 2）と RFOB 実装（Step 3）は並行作業であり、実測値は Step 3 で初めて得られる
- 概算値でも理論の検証（安全集合の前方不変性、Graspability Index の挙動）には十分
- config/objects/ の JSON を差し替えるだけで実測値に移行可能

## 物体パラメータ一覧（概算値）

| 物体 | K_0 [Nm/rad] | θ_yield [rad] | β [1/rad] | 質量 [kg] | τ_min [Nm] | 特性 |
|------|-------------|---------------|-----------|----------|------------|------|
| 空アルミ缶 | 5.0 | 0.03 | 50 | 0.015 | 0.001 | 硬い、軽い |
| PETボトル（水入り） | 3.0 | 0.05 | 30 | 0.500 | 0.025 | 中剛性、重い |
| PETボトル（空） | 0.5 | 0.02 | 40 | 0.030 | 0.002 | 柔らかい、軽い |
| 紙コップ（空） | 0.2 | 0.01 | 80 | 0.010 | 0.001 | 非常に柔らかい |
| 紙コップ（水入り） | 0.3 | 0.015 | 60 | 0.200 | 0.010 | 柔らかい+重い（最も厳しい） |

**τ_min の概算式**:
```
τ_min = (m × g × r_eff) / (2 × μ)
```
- g = 9.81 m/s²
- r_eff ≈ 0.02 m（指先の有効半径、概算）
- μ ≈ 0.8（ゴム指先と物体表面の摩擦係数、概算）

**注意**: これらは全て概算値。Step 3 の RFOB 実機検証で K_0, θ_yield, β を実測し、config の JSON を差し替える。
